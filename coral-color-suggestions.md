# Coral Palette — Same Tone as `#FF6B5C`

:::callout{type=info title="Base color: Coral — #FF6B5C"}
The current **Operators / brand** color in Tropical Sorbet (light). All suggestions below share the same warm, saturated tropical tone. The existing shade `#E55D4E` is already used as `statusBar.border`.
:::

---

## Lighter — Highlights & Hover States

:::cards{cols=3}
:::card{title="#FF8A7A" icon=palette}
:chip[Soft coral]{color=warning}

Gentle hover background, badge fill, or selection tint. Close to the base but clearly lighter.
:::
:::card{title="#FF9E90" icon=palette}
:chip[Warm salmon]{color=warning}

Works well for subtle chip backgrounds or notification badges — readable on cream `#FFF6EC`.
:::

:chip[Coral blush]{low}

Very light — ideal as a faint tinted background (e.g. diff-added, highlighted row).
:::
:::

## Shifted Hue — Same Family, Different Direction

All colors below share the same saturation and lightness as `#FF6B5C` (HSL 5°, 100%, 68%). Only the hue rotates.

### Spectrum overview

| ← Pink | | Base | | Orange → |
|--------|--|------|--|----------|
| `#FF5CBB` | `#FF5C92` | `#FF6B5C` | `#FF925C` | `#FFBB5C` |
| −40° | −25° | 0° | +15° | +30° |
| :chip[Coral-magenta]{color=danger} | :chip[Warm pink]{color=danger} | :chip[Coral]{color=warning} | :chip[Coral-orange]{color=warning} | :chip[Warm orange]{color=warning} |

---

### Toward Watermelon `#E5397A` — pink direction

:::cards{cols=3}
:::card{title="#FF5C77 — −15°" icon=palette}
:chip[Pink-coral]{color=danger}

Barely shifts from the base. A hint of pink, still reads as coral. Good bridge for a gradient or hover transition.
:::
:::card{title="#FF5C92 — −25°" icon=palette}
:chip[Warm pink]{color=danger}

Noticeably pinker. Sits comfortably between coral and Watermelon — useful as a secondary keyword accent.
:::
:::card{title="#FF5CBB — −40°" icon=palette}
:chip[Coral-magenta]{color=danger}

Strong pink shift. Approaches Watermelon territory — use intentionally, not as a neutral.
:::
:::

:::cards{cols=3}
