# 🍧 Tropical Sorbet

**Bright coral & citrus candy on cream.** Two carefully crafted themes for long, happy coding sessions.

---

## Themes

| Variant | Base | Mood |
|---------|------|------|
| **Tropical Sorbet** | `#FFF6EC` warm cream | Bright, airy, day session |
| **Tropical Sorbet Night** | `#2B1E1B` warm cocoa | Vibrant candy on dark chocolate |

## Install

**From the Marketplace:**
1. Open VS Code
2. Press `Ctrl+P` (macOS: `Cmd+P`)
3. Paste: `ext install ChieuLe.tropical-sorbet`
4. Press Enter

**Activate:**
Press `Ctrl+K Ctrl+T` and pick **Tropical Sorbet** or **Tropical Sorbet Night**.
